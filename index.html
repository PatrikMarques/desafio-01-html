O que é HTML?

(Voltar)

HTML é a sigla para HyperText Markup Language (Linguagem de Marcação de Hipertexto). É a linguagem padrão usada para criar páginas da web. Em outras palavras: O HTML serve para estruturar o conteúdo de um site. Ele organiza textos, imagens, vídeos, botões, tabelas e outros elementos que aparecem numa página.

Quais tags eu aprendi

(Voltar)

Utilizar todas as tags explicadas nas aulas: h1 até h6, p, mark, small, i, u, strong, ol, ul, li, a, hr, sub, sup, blockquote. Essas são tags básicas do html, porém necessárias para criar a base de qualquer site ou blog em html.

Novas tags que aprendi

(Voltar)

font Obsoleta: Essa tag não é mais recomendada no HTML5. Ela era usada para definir cor, tipo e tamanho da fonte do texto. Hoje, isso é feito com CSS.

del Serve para riscado ou indicar que um texto foi removido (como se fosse uma edição).

abbr Usada para mostrar abreviações, com a explicação aparecendo quando você passa o mouse por cima (tooltip).
